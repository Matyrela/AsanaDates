Acerca de este programa

Este programa, usando su token personal, accede a sus proyectos y las tareas de los mismos guardando los IDs en el archivo config.conf, con estos IDs luego se piden datos como las modificaciones de fecha de entrega, los IDs de los proyectos se guardan para que en la siguiente ejecución del código el programa evite hacer llamadas a la API que no son relevantes, ya que la misma tiene un límite.
Es importante, que si se crea un nuevo proyecto en Asana, deberás apretar el botón de borrar datos, así el programa borrara los IDs guardados y pedirá a la API los proyectos del usuario incluyendo los nuevos.
